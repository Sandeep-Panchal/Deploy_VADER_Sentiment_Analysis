VADER Sentiment Analysis

'VADER Sentiment Analyzer' takes in the text of the user and predicts whether a entered text is a Positive, Negative or Neutral.

Prediction of the entered text is based on the packageVADER Sentiment Analyzer. You can visit the blog Sentiment Analysis with VADER- Label the Unlabelled Data that I have written.

The main purpose of this project is to learn the deployment in AWS EC2 Instance. Hence, I have used the package. Soon, I will come up with some predictive model on sentiment analysis :)

This is not for any business purpose. This is 'strictly' for a 'learning' purpose.
